# ☕ IFPI Java Studies

Repositório dedicado aos meus estudos, exercícios e projetos desenvolvidos em **Java** durante minha formação no **Instituto Federal do Piauí (IFPI)**.

Este repositório documenta minha evolução na linguagem Java desde o **Curso Técnico em Informática** até a graduação em **Análise e Desenvolvimento de Sistemas (ADS)**, reunindo atividades acadêmicas, projetos práticos e implementações de Programação Orientada a Objetos.

---

## Objetivos

- Consolidar os conhecimentos em Java;
- Praticar Programação Orientada a Objetos (POO);
- Desenvolver projetos acadêmicos;
- Organizar exercícios realizados durante a formação;
- Documentar minha evolução como desenvolvedor;
- Construir um portfólio de projetos em Java.
